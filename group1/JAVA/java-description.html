Über Java:

Java ist eine Objektorientierte Programmiersprache,
die ursprünglich von
Sun Microsystems entwickelt wurde und heute eine eingetragene Marke
von Oracle ist.

Die Programmiersprache Java beeinhaltet:

Java Development Kit (JDK)
Java Runtime Environment(JRE)
Java Virtual Machine (JVM)

JDK:
Das JDK beinhaltet neben der Laufzeitumgebung Java Runtime Environment (JRE) noch weitere tools

z.B.
Java-Compiler
Java-Debugger

Klassendiagramm:

- Klasse:

Eine Klasse beschreibt
die Struktur und das Verhalten von Objekten, die auf Basis
einer Klasse erstellt werden.

In einer Klasse werden Eigenschaften (Attribute) und Funktionen (Methoden definiert).

- Klassendiagramm:

Ein Klassendiagramm dient zur grafischen Darstellung von Klassen und deren Beziehungen zu einander.

Das Klassendiagramm gehört zu Strukturdiagrammen der Unified Modeling Language (UML).

Anwendungen:

  • Mobile Apps
  • Desktop Apps
  • Webbasierte Apps
  • Spiele
  • IoT
  • Cloud-basierte Apps

HTML oder auch Hypertext Markup Language ist der erste Schritt den man nehmen muss. Man bezeichnet HTML auch als: Die grundlegende Struktur und Inhalte einer Website. Hierbei werden Strukturen gebildet welche als Grundlage der Website dienen.

CSS

HTML hat also deine Interessen geweckt und du willst mehr zur Webentwicklung lernen? Dann lernst du jetzt was über CSS. CSS (Cascading Style Sheets) dient zur Gestaltung von HTML-Elementen, einschließlich Farben, Schriftarten und Layouts. CSS gibt der Website einfach gesagt Farbe und einen Charakter.
